# Computer-Vision

This repository contains a collection of computer vision projects that I worked on using Python and OpenCV. These projects helped me understand core image processing concepts such as feature detection, image transformation, object detection, and face recognition.

Technologies Used
Python, OpenCV, NumPy, Matplotlib, Jupyter Notebook

1. Computer Vision Basics
File: Computer Vision.ipynb
This notebook contains foundational computer vision operations such as: Image loading, Image filtering, Edge detection, Image transformations, Basic preprocessing techniques
This helped build the foundation before working on advanced projects.

2. Hough Circle Detection & Harris Corner Detection
File: Hough circle_Harris corner.ipynb
This notebook demonstrates two important feature detection techniques.
 Hough Circle Detection - Used to identify circular objects in images
 Harris Corner Detection - Used to detect corners and interest points in images.

3. Face Recognition
File: Face Recognition.ipynb
This project performs face recognition using image processing techniques.

4. Image Stitching
File: Image stitching.ipynb
This project combines multiple overlapping images into a single panoramic image.
